# Katachi for Developers 利用規約

最終更新: 2026-05-09

この利用規約は、Katachi for Developers（以下「本アプリ」）の利用条件を定めるものです。ユーザーは、本アプリの利用開始前に本規約およびプライバシーポリシーへ同意する必要があります。同意しない場合、本アプリを利用できません。

## 1. 本アプリについて

本アプリは、開発中の着想、実装メモ、検証メモなどをカードとして保存し、あとで再利用しやすくするための補助ツールです。

本アプリは、保存内容の正確性、完全性、継続保存、特定目的への適合性、第三者権利の非侵害、または将来のOSや端末での動作を保証しません。

ユーザーは、次の内容を本アプリに保存または利用してはいけません。

- 第三者の著作権、商標権、営業秘密、プライバシーその他の権利を侵害する内容
- 所属組織や取引先との秘密保持義務に反する内容
- 違法行為、危険行為、権利侵害を助長する内容
- 不正アクセス、マルウェア、脆弱性悪用を目的とした内容
- その他、開発者が不適切と判断する内容

## 3. 保存データとバックアップ

Freeプランでは、カード内容は原則として端末内に保存されます。Plus / Proでクラウド同期を利用する場合、認証情報、カード内容、タグ、作成/更新日時、購入状態などをFirebaseに保存することがあります。

ユーザーは、重要な情報を自己の責任でバックアップしてください。アプリの不具合、同期失敗、端末故障、誤操作、アカウント削除、サービス終了などによるデータ消失について、法令で認められる最大限の範囲で、開発者は責任を負いません。

## 4. 有料機能

Plus / Proは自動更新サブスクリプションです。

- Plus: 月額460円 / 年額4,600円
- Pro: 月額1,200円 / 年額12,000円

購入はApple IDに請求されます。現在の期間終了24時間前までに解約しない限り、自動更新されます。サブスクリプションの管理、解約、返金は、App Storeの規約およびAppleの手続きに従います。

Freeプラン向けに、消耗型アイテムとして追加保存枠を提供することがあります。

- 100円: +25件
- 300円: +100件
- 500円: +200件

追加保存枠はFreeプランのローカル保存上限に加算されます。
