## User Privacy Choices URL用

### Katachi for Developers データ管理と削除

Katachi for Developersでは、Freeプランのカード内容は原則として端末内に保存されます。Plus / Proでクラウド同期を利用する場合、GoogleログインまたはSign in with Appleのアカウントに紐づけて、カード内容をFirebaseに保存します。

ユーザーができること:

- アプリ内でカードを削除する
- 設定タブからログアウトする
- 設定タブのクラウド同期メニューからクラウドアカウント削除を実行する
- 削除に失敗した場合、サポート窓口へ削除依頼を送る

削除依頼時に必要な情報:

- アプリ名: Katachi for Developers
- 利用していたログイン方法: Google / Apple
- ログインに使ったメールアドレス、またはAppleの非公開メールアドレス
- 削除したい内容: クラウドアカウント、カードデータ、問い合わせ履歴など

サポート:

- `https://anemos-dev.github.io/KatachiForDevelopers/support.html`
- `kazahaworkspace@gmail.com`

## Support URL用

### Katachi for Developers サポート

お問い合わせ前に確認してください。

- 購入、解約、返金はApp StoreのApple ID管理から行います
- Plus / Proのクラウド同期にはログインが必要です
- Freeプランの保存データは端末内保存のため、端末削除や故障で復元できない場合があります
- 重要なメモは共有機能でバックアップしてください

問い合わせ内容に含めると対応しやすい情報:

- 使用端末
- iOSバージョン
- アプリバージョン
- 発生した画面
- 再現手順
- スクリーンショット

サポート:

- `https://anemos-dev.github.io/KatachiForDevelopers/support.html`
- `kazahaworkspace@gmail.com`
