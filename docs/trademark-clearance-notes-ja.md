# Katachi 商標・同名アプリ一次調査メモ

最終更新: 2026-05-04

## 結論

`Katachi` は完全に空いている名称とは言い切れない。初回リリースは `Katachi for Developers` のフルネームを前面に出し、ホーム画面の短縮名 `Katachi` は維持してもよいが、商標出願前提なら弁理士確認を推奨する。

特に、Kashika, Inc. による `KATACHI` のソフトウェア/3Dデータ関連の情報が見つかっている。米国USPTOの `97288982` は2023-04-03時点で放棄扱いだが、EUIPO系の `018659110` は登録状態として確認できる。今回のアプリは開発者向けメモ/生産性ツールなので用途は離れているが、区分9/42のソフトウェア領域が重なる可能性はある。

## 見つかった近似名称

- App Store: `katachi` という予約系アプリが日本ストアに存在。カテゴリはライフスタイルで、販売元は Masahide Kondo。
- App Store: `katachi-シェイプタップで脳トレ` というパズル/脳トレ系アプリが存在。
- USPTO/周辺DB: `KATACHI`、Serial Number `97288982`、出願人 Kashika, Inc.。3Dデータ共有、3Dモデリング、クラウド/ソフトウェア関連の指定商品・役務。2023-04-03に `Abandoned - Office Action Response Not Filed or Filed Late` と表示。
- EUIPO周辺DB: `Katachi`、Application Number `018659110`、出願人 Kashika, Inc.。3Dデータ/ソフトウェア関連。2022-08-01にRegisteredと表示。

## 推奨

- App Store表示名は `Katachi for Developers` を使い、単独の `Katachi` だけでマーケティングしない。
- スクリーンショット、Web、説明文では「開発アイデア管理」「developer productivity」を明確にし、3D/CAD/形状データサービスに見えない表現にする。
- 日本、米国、EUで正式に商標検索し、必要なら `Katachi for Developers` と将来の `Katachi for Designers` を含めて出願戦略を決める。
- `Katachi` 単独の出願・独占が難しそうなら、`Katachi Dev`、`Katachi Studio` など識別力を足した候補も検討する。
