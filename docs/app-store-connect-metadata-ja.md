## 説明文

Katachi for Developersは、開発者のためのアイデア保存アプリです。

作りたい機能、直したい違和感、あとで検証したい技術、記事ネタ、バグ仮説などをカードとして保存し、グループ、種類、ステータス、タグ、期限、次にやることと一緒に整理できます。

主な機能:

- 思いついたらすぐ保存できるクイック保存
- グループ、種類、ステータス、タグによる整理
- お気に入り、検索履歴、期限メモ
- Markdown、GitHub Issue、レビュー用メモとして共有
- Freeでは端末内に保存
- Plus / ProではGoogleまたはAppleログインによるクラウド同期
- Free向けの買い切り追加保存枠

Freeプランでは端末内に50件まで保存できます。Plus / Proでは、機種変更や複数端末利用に備えてクラウド同期を利用できます。

自動更新サブスクリプションについて:

- Plus: 月額460円 / 年額4,600円
- Pro: 月額1,200円 / 年額12,000円
- 購入はApple IDに請求されます
- 現在の期間終了24時間前までに解約しない限り、自動更新されます
- サブスクリプションの管理と解約はApp Storeのアカウント設定から行えます

Free向け追加保存枠:

- 100円: +25件
- 300円: +100件
- 500円: +200件
